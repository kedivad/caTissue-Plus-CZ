About caTissue Suite v1.1.p
caTissue Suite v1.1.p is brought to you by Persistent Systems, a NCI certified caBIG Support Service provider. This version of caTissue is free to use and is an open source application. The new features in this release are either funded by Persistent or by our various customers. We thank all our customers for their generosity in allowing us to use their application extensions and making it freely available to the rest of the community.
Disclaimer: This is not an official NCI release of caTissue. Please contact us on 1-877-CATISSU or e-mail us at catissue_support@persistent.co.in for any details about this release.

Features of caTissue Suite v.1.1.p1
 
The major features in the caTissue Suite v1.1.p1 release were:
  • Search functionality like Google
  • Support for UK locale
  • Auto generation capability for PPI field
  • Ability to create Collection protocol with no specimen requirements
  • Auto complete dropdowns for protocols and participants in Collection Protocol based view
  • Improved Collection Protocol based view for specimen data entry
  • Possible to Show/hide Dynamic Extensions forms and entities on specific Collection Protocols
  • Advance query interface support outer join
  • Fluid UI support for all screen resolutions
  • Ability to a select printer location and printer type while label printing
  • Print functionality for Storage container labels
  • Performance enhancements
  • Provision for #54 bug fixes
